# **App Name**: Mengs

## Core Features:

- Google Authentication: Login using Google OAuth2.
- Course Management: Manage semesters and courses. Creating, reading, updating, and deleting semesters and courses.
- Task Management: Manage tasks with deadlines, status, and priority. CRUD operations are fully supported.
- Push Notifications: Send push notifications (PWA) and emails for upcoming deadlines.
- Automated File Sync: Automatically synchronize files with Google Drive in a structured academic hierarchy.
- Calendar Integration: Integrate with Google Calendar for optional schedule synchronization.
- Schedule Import: Import schedules from CSV or ICS files for easy setup.
- Schedule Templates: Offer schedule templates for quick semester planning.
- Dark Mode: Offer a dark mode option for comfortable use in low-light environments.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to reflect intellect and focus.
- Background color: Very light indigo (#F0F2FA), almost white, for a clean and calm study environment.
- Accent color: Blue (#2979FF), a brighter analogous color, used for highlights and calls to action to guide the user.
- Headline font: 'Jakarta Sans' sans-serif, for titles. A modern, clean font well-suited for section headings and important information.
- Body font: 'Inter' sans-serif, for the bulk of textual content. Ensure comfortable reading.
- Use consistent and clear icons from a set like 'lucide' to represent different course types, tasks, and priorities. Use simple, minimalist icons for a modern and uncluttered feel.
- Mobile-first, responsive layout to work on both desktop and mobile devices.
